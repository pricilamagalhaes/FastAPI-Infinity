# Iniciando Projeto FastAPI

### Ambiente virtual

- Windows
```bash
py -m venv ./nomedapasta
```

- Linux

```bash
python3 -m venv ./nomedapasta
```

#### Iniciando ambiente

- Windows
```bash
./nomedapasta/Scripts/Activate.ps1
```

- Linux

```bash
source ./nomedapasta/Bin/activate
```
